Propiedad position

position: static (valor por defecto)

Static

El elemento sigue el flujo normal del documento. Las propiedades top, right, bottom, left y z-index no tienen efecto.

position: relative

Relative

El elemento se posiciona relativamente a su posición normal. Puedes moverlo usando top, right, bottom, left.

position: absolute

Contenedor relativo
Absolute

El elemento se posiciona relativamente al padre posicionado más cercano (no static). Si no hay ninguno, se usa el body.

position: fixed

El elemento (cuadro morado) está fijo en la ventana del navegador, no se mueve al hacer scroll. Mira abajo a la derecha.

position: sticky

El encabezado de esta sección tiene position: sticky. Se comporta como relative hasta que alcanza la posición top definida, entonces se queda "pegado".

Propiedad display

display: block

Block

Ocupa todo el ancho disponible y fuerza un salto de línea antes y después.

display: inline

Texto normal Inline continúa en la misma línea. No se puede definir width ni height.

display: inline-block

Inline-Block

Se muestra en línea pero acepta propiedades de bloque como width y height.

display: none

Hidden

El elemento no se muestra y no ocupa espacio en el layout.

visibility: hidden

Visibility

El elemento no es visible pero sigue ocupando espacio.
